# Net Promoter Score Analysis Using NumPy

This project analyzes customer feedback data and calculates the Net Promoter Score (NPS) using Python and NumPy.

## Skills Demonstrated
- NumPy arrays
- Boolean masking
- Data aggregation
- Business analytics
- Function creation
- Vizualization using matplotlib
  
## Technologies Used
- Python
- NumPy
- Matplotlib
- Jupyter Notebook
